## Day 1
### HTML이란?
  Hyper Text Markup Language의 약자이며, 태그를 이용한 Markp언어이다.
### HTML Tag
  특징, 구조<br/>
  시작(태그) - 컨텐츠 - 종료(태그)
### HTML 문서 기본구조
  doctype, html, head, body<br/>
  html태그로 시작하며, head는 문서 정보를 나타내고, body는 실제 보여지는 부분
### 주석
  `<!-- -->` 이렇게 표시함<br/>
  주석처리할 부분 선택 후 `ctrl or command + /` 로도 가능
